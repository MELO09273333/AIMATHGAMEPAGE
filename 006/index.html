遊戲介紹

Black Path Game（又稱Brick）是一款兩人對戰的棋盤遊戲。玩家輪流在棋盤上延伸路徑，目標是將路徑連接到終點（基地），或讓對手無法繼續。

基本規則

  • 遊戲在m×n方格棋盤上進行，左下為起點，右上為終點。
  • 每回合，玩家必須在延續路徑的空格放置一個連接片（T1、T2、T3）。
  • 連接片有三種：

T1

T2

T3
  • 誰先將路徑連接到終點（基地）即獲勝。
  • 若路徑連接到棋盤邊界（出界），則該玩家輸。

操作說明

  • 選擇棋盤大小，點擊「開始」開始遊戲。
  • 拖曳下方的連接片到下一步的位置。
  • 無誤後，點擊「OK」，換對方操作。
  • 可用「上一步」「下一步」瀏覽歷程。
  • 右上角「重置」可重新開始。

進階說明

  • 若路徑遇到已放片的格子，會自動沿著連接片推進，直到遇到空格、終點或邊界。
  • 詳細規則與數學分析可參考 Black Path Game 維基百科。
